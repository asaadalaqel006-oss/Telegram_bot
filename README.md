# Telegram Moderation Bot

بوت تيليجرام جاهز لإدارة المجموعات وحذف الرسائل الإعلانية والكلمات المسيئة، مع **حظر تلقائي** لمن يرسل الإعلانات أو الروابط أو أرقام الهواتف، بالإضافة إلى **ردود تلقائية** عند إرسال كلمات محددة مثل: `ايفولف 2` و `ايفولف 3` و `ايفولف 4`.

## المميزات

- حذف الرسائل الإعلانية تلقائيًا
- حظر المستخدم الذي يرسل إعلانًا أو رابطًا أو رقم هاتف
- حظر الرسائل التي تحتوي على:
  - روابط
  - يوزرات مثل `@username`
  - أرقام هواتف
  - عبارات إعلانية وتعليمات تسويقية وخدمات طلابية
- حذف الكلمات المسيئة
- إمكانية جعل الكلمات المسيئة تؤدي إلى الحظر بعد عدد معين من المخالفات
- استثناء مشرفي الجروب تلقائيًا من الفلترة
- دعم قائمة استثناء إضافية عبر `SAFE_USER_IDS`
- رد تلقائي على كلمات محددة بروابط جاهزة
- إعدادات مناسبة للتشغيل على Railway

## الردود التلقائية

### عند إرسال: `ايفولف 2`
يرد البوت بهذه الروابط:

- كتاب الطالب 2

- حلول اخر الكتاب
- نصوص استماع كتاب الطالب
- نصوص استماع كتاب النشاط
- حلول كامبردج 2
- كتاب النشاط
- حلول كتاب النشاط

### عند إرسال: `ايفولف 3`
يرد البوت بهذه الروابط:

- الكتاب الطالب 3
- كتاب المعلم
- حلول نهاية الكتاب
- حلول كامبردج 3

### عند إرسال: `ايفولف 4`
يرد البوت بهذه الروابط:

- كتاب ايفولف 4 كامل
- حلول اخر الكتاب
- كتاب النشاط
- حلول كتاب النشاط
- حلول الكامبردج
- نصوص استماع

## المتطلبات

- Python 3.10+
- توكن بوت تيليجرام من BotFather

## الإعداد السريع محليًا

### 1) تثبيت المكتبات

```bash
pip install -r requirements.txt
```

### 2) تجهيز ملف البيئة

```bash
cp .env.example .env
```

ثم ضع التوكن داخل `.env`:

```env
BOT_TOKEN=YOUR_TELEGRAM_BOT_TOKEN
```

### 3) تشغيل البوت

```bash
python main.py
```

## التشغيل على Railway

المشروع مجهز بملف `railway.json` و `Dockerfile`.

### خطوات النشر

1. ارفع المشروع إلى GitHub.
2. افتح Railway.
3. أنشئ مشروع جديد من GitHub repo.
4. اختر المستودع.
5. أضف متغير البيئة التالي داخل Railway:

```env
BOT_TOKEN=YOUR_TELEGRAM_BOT_TOKEN
```

6. نفّذ النشر وسيبدأ Railway تشغيل البوت تلقائيًا.

### متغيرات اختيارية مفيدة على Railway

```env
MODERATION_ENABLED=true
BLOCK_LINKS=true
BLOCK_PHONE_NUMBERS=true
SEND_ALERT_MESSAGES=true
ALERT_MESSAGE_AUTO_DELETE_SECONDS=8
ABUSE_BAN_THRESHOLD=0
DROP_PENDING_UPDATES=true
SAFE_USER_IDS=123456789,987654321
```

## مهم جدًا داخل تيليجرام

حتى يعمل البوت على حذف الرسائل وحظر الأعضاء، يجب أن:

1. تضيفه كمشرف داخل المجموعة.
2. تمنحه صلاحيات:
   - Delete messages
   - Ban users
3. توقف **Privacy Mode** من BotFather حتى يتمكن من قراءة رسائل الأعضاء داخل المجموعة.

## الأوامر

- `/start` : التحقق من أن البوت يعمل
- `/ping` : اختبار سريع
- `/status` : عرض حالة الإعدادات

## تخصيص الكلمات المسيئة

افتح ملف `settings.py` وأضف الكلمات المسيئة داخل:

```python
CUSTOM_BANNED_ABUSE_TERMS = [
    "كلمة1",
    "كلمة2",
]
```

## التحكم في الحظر عند الكلمات المسيئة

داخل ملف `.env` أو متغيرات Railway:

```env
ABUSE_BAN_THRESHOLD=0
```

- إذا كانت القيمة `0` فلن يتم الحظر بسبب الكلمات المسيئة، وسيتم فقط حذف الرسالة.
- إذا جعلتها مثلًا `3` فسيتم حظر المستخدم بعد 3 مخالفات إساءة.

## التشغيل عبر Docker

```bash
docker build -t telegram-moderation-bot .
docker run --env-file .env telegram-moderation-bot
```
